# Cube Timer and Scrambler
- An application built on the Adobe AIR runtime with ActionScript 3.0 to practice Rubik's Cube speedsolving on an iOS device

## Features
- Tap to start timer, tap again to stop timer
- Automatically generates random scramble on stop
